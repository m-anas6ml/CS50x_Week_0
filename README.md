This a simple game in which you:
  1. Drag the pointer around to catch the fruit.
  2. The Goal is to reach 31 points.
  3. Each fruit carries +1 point if caught and -1 if not caught,
  4. except fruit Salad it carries +2 and -2 points.

This project was creadted to understand how the basic of coding work, which included:
  1. Loops
  2. Variables
  3. Functions
  4. etc

You are welcome to try the game at the given link and use it as you see fit.
